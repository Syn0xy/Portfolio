Voici la liste de tous mes projets personnels, ainsi que toutes les technologies que je maîtrise.

TYPESCRIPT

Développé par Microsoft, ce langage est un sur-ensemble de JavaScript qui a pour but d'améliorer et sécuriser le code.

JAVASCRIPT

Le JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives, il est une partie essentielle des applications web.

C#

Le C# est un langage de programmation orientée objet, dérivé de C et de C++, ressemblant au langage Java.

JAVA

Le Java est un language de programmation orientée objet, il reprend en grande partie la syntaxe du C++ en simplifiant la gestion des notions, des pointeurs, etc.

CSS

CSS ou Cascading Style Sheets, est un langage informatique qui décrit la présentation des documents HTML.

HTML

HTML ou HyperText Markup Language, est utilisé afin de créer et de représenter le contenu d'une page web et sa structure.

SQL

Le SQL (Structured Query Language) permet de manipuler les données et les systèmes de bases de données relationnelles.

C++

C++ est un langage de programmation compilé, permettant la programmation sous de multiple forme, dont la programmation procédurale, la programmation orientée objet et la programmation générique. C'est un des langages de programmation les plus utilisés dans les applications où la performance est critique.

BASH

Bash ou Bourne-Again Shell, est un interpréteur en ligne de commande de type script.

C

C est un langage de programmation impératif, généraliste et de bas niveau. Inventé en 1972, il est devenu un des langages les plus utilisés.
